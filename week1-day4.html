    • Come creare un template per i post

    Di Mario Rossi - pubblicato il 25-10-2020

    La prima cosa da fare è suddividere in blocchi la pagina, identificando le parti comuni da applicare a tutte le pagine

    Le varie aereSe saranno quindi suddivise in:
  • Contenuti di intestazione del tag < header >
  • Contenuti centrali variabili del tag < main > { corpo principale }
  • Contenuti di fine pagina comuni, da inserire nal tag < footer >

    • Formattazione del codice Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con CSS. Perciò i passi sarannno:

    • Progettare il markup assegnando ad ogni specifco testo il tag appropriato
      • Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
        • Applicare gli stili CSS

        L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale.
